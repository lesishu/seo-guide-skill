# 内容策略与 SEO 写作

## 内容策略框架

### 1. 内容漏斗与对应内容类型

```
认知层（Top of Funnel）
  → 目的：吸引访客、建立品牌认知
  → 类型：博客文章、社交媒体内容、信息图表、视频
  → 关键词：信息型、高搜索量、低商业意图

考虑层（Middle of Funnel）
  → 目的：展示专业性、建立信任
  → 类型：对比评测、案例研究、白皮书、指南
  → 关键词：商业型、中等竞争

决策层（Bottom of Funnel）
  → 目的：推动转化
  → 类型：产品页、定价页、FAQ、CTA
  → 关键词：交易型、高转化意图

保留层（Retention）
  → 目的：提升 LTV、复购
  → 类型：使用教程、技巧博文、邮件 newsletter
```

### 2. 内容主题发现

**来源：**
- 用户真实问题（客服工单、评论、社交媒体）
- Google Search Console 中表现好的查询
- "人们也在问"（PAA，People Also Ask）
- 竞品内容差距分析
- 行业趋势和新闻

### 3. SEO 友好文章结构

**标题（H1）：** 包含主关键词 ≤ 60 字符，数字开头或疑问句通常表现好
**URL：** `/主关键词-副关键词/`
**Meta 描述：** 150-160 字符，含 CTA

**正文结构：**
```
导言（100-200字）
  → 描述问题/场景
  → 告诉读者会学到什么
  → 包含主关键词

正文主体（H2-H3）
  → 每个 H2 回答一个子问题
  → 使用项目符号/编号列表
  → 包含相关长尾关键词

FAQ（H2）
  → 来自 People Also Ask
  → 用 FAQ Schema 标记（可触发精选摘要）

总结（H2）
  → 核心要点回顾
  → 包含主关键词的变体
  → CTA（视情况添加）
```

### 内容质量原则（People-First Content）

**Google 明确说：** "Creating content that people find compelling and useful will likely influence your website's presence in search results more than any of the other suggestions in this guide."（让用户觉得有用、有趣的内容，对搜索排名的影响力超过本指南中的任何其他建议。）

高质量内容共同特征：
- **易于阅读**：自然写作、无拼写/语法错误，有标题分段
- **内容独特**：不复制他人内容，基于自己的知识和经验原创
- **内容最新**：定期检查并更新旧内容，无相关内容则删除
- **以人为本**：提供有帮助、可信赖的内容

### E-E-A-T 澄清（重要）

> ⚠️ **Google 官方明确说明：E-E-A-T 不是排名因素。** E-E-A-T 是 Google 质量评估员用来评估内容质量的**框架**，用于判断内容是否 Helpful（有帮助）。

与其刻意"满足 E-E-A-T"，不如：
- 分享真实的第一手使用经验（Experience）
- 引用专业数据源和有经验的信息源（Expertise）
- 链接到权威网站，建立权威性（Authoritativeness）
- 提供透明联系方式、用户评价（Trustworthiness）

### 5. 文章长度与深度

- **没有固定字数要求** — 以完整回答用户问题为准
- 竞品 Top 10 平均深度是基准
- 深度覆盖用户问题越多，排名潜力越大
- 每 500-800 字插入一个小标题（H3）

### 6. 内容更新策略

**定期审计：**
- 季度检查高流量文章
- 更新过时信息（价格、功能、统计数据）
- 添加新的相关关键词
- 修正失效的外部链接

**更新信号：** 修改日期（`article:modified_time`）、显著的内容增改

### 7. 内容差异化

- 独家数据或调研（自己做的问卷/分析）
- 独特的写作视角或风格
- 更丰富的多媒体（视频、交互工具）
- 更详细的操作步骤（截图、代码）
- 本地化/行业化定制

### 8. 多媒体策略

| 类型 | SEO 价值 | 制作成本 |
|------|---------|---------|
| 截图/截图+GIF | 展示操作过程，高分享率 | 中 |
| 信息图（Infographic）| 高外链潜力、社交分享 | 高 |
| 视频（嵌入 YouTube）| 增加页面停留时间 | 高 |
| 数据图表 | 增强可信度、容易被引用 | 中 |
| 可下载模板/清单 | 收集邮箱（增长转化）| 中 |

### 9. 避免常见 SEO 内容错误

- 标题与内容不符（标题党）
- 关键词堆砌（密度 > 5%）
- 内容单薄（100-200 字无法满足需求）
- 没有内链（孤立页面）
- 外部链接失效（定期检查）
- 大量重复内容（每个页面解决不同问题）
