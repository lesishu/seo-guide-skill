# 关键词研究与分析

## 关键词研究流程

### 1. 确定种子关键词
从业务核心出发，列出 5-10 个种子词。

**方法：**
- 产品/服务名称（如"项目管理软件"）
- 用户痛点（如"任务协作"）
- 行业术语（如"Scrum敏捷"）
- 竞品品牌词

### 2. 扩展关键词列表

**工具：** Google Keyword Planner / Ahrefs / Semrush / Ubersuggest

**扩展方法：**
- 种子词变体：同义词、复数形式、拼写变体
- "人们也在搜"（People Also Search For / People Also Ask）
- Google 自动补全（下拉建议）
- AnswerThePublic 的问题词（who/what/why/how/where）

**示例**（种子词：项目管理软件）：
- 项目管理工具、项目追踪软件、团队协作软件
- 任务管理工具、OKR管理、甘特图软件
- 远程团队协作、项目管理APP

### 3. 分析关键词指标

| 指标 | 说明 | 理想值 |
|------|------|--------|
| 搜索量（Search Volume） | 每月平均搜索次数 | 100-1000（长尾词）|
| 关键词难度（KD） | 排名前10的难度 | <30（新人站）/ <50（有权重站）|
| CPC（每次点击成本） | 竞价广告价格 | 间接反映商业价值 |
| 搜索意图（Intent） | 导航/信息/商业/交易 | 根据目标选择 |

### 4. 搜索意图分类

| 类型 | 用户意图 | 内容类型 |
|------|---------|---------|
| **导航型**（Navigational） | 找特定网站 | 品牌页、官网 |
| **信息型**（Informational） | 获取信息/答案 | 博客文章、指南、FAQ |
| **商业型**（Commercial） | 研究购买前的选项 | 对比文章、评测 |
| **交易型**（Transactional） | 完成购买/转化 | 产品页、定价页、CTA |

**优先覆盖顺序：** 信息型 → 商业型 → 交易型（内容营销阶段）→ 导航型（品牌词）

### 5. 关键词分层策略

```
核心词（大词）：竞争激烈 → 放首页/栏目页 → 需要高权重支撑
次要词（中等）：中等竞争 → 放分类页/专题页
长尾词（长尾）：低竞争 → 放文章页/问答页 → 容易获得排名
```

### 6. 关键词分组与映射

将关键词按主题/意图分组，为每个分组规划对应页面。

**示例：**
| 主题群 | 目标关键词 | 目标页面类型 |
|--------|-----------|------------|
| 项目管理软件基础 | 什么是项目管理软件、项目管理软件推荐 | 博客文章 |
| 项目管理功能 | 任务管理功能、甘特图工具、团队协作功能 | 产品功能页 |
| 项目管理教程 | 如何做项目管理、项目管理流程图 | 教程/指南页 |
| 项目管理软件对比 | Trello vs Asana、项目管理工具对比 | 对比评测页 |

## 竞品关键词分析

1. 找到竞争对手（Google 搜索核心词，Top 10 中是谁）
2. 用 Ahrefs/Semrush 导出竞品的有机关键词
3. 找出竞品有排名但自己尚未覆盖的关键词
4. 评估这些词的搜索量和难度，确定优先级
5. 分析竞品内容策略：他们写了什么？内容深度如何？格式是什么？

## 关键词密度与自然融入

- **密度建议：** 2-5%（自然出现，不刻意堆砌）
- **标题（H1）：** 必须包含主关键词
- **副标题（H2-H3）：** 包含相关变体词
- **首段（100-150字）：** 核心关键词出现一次
- **正文：** LSI 关键词（语义相关词）自然分布
- **结尾：** 再次出现主关键词
