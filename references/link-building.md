# 外链建设（Link Building）

## 外链基础概念

| 指标 | 说明 |
|------|------|
| DA（Domain Authority）| Moz 开发的域名权威分数 1-100 |
| DR（Domain Rating）| Ahrefs 的域名评分，0-100 |
| SPAM Score | Moz 的垃圾链接风险评分，越低越好 |
| Dofollow 链接 | 传递 PageRank（权重），高价值 |
| Nofollow 链接 | 不传递权重，但仍有引荐流量和品牌曝光价值 |
| UGC 链接 | 评论、论坛等自动加 nofollow |
| Sponsored | 广告/赞助内容链接 |

## 白帽外链策略

### 1. 数字/原创研究（Skyscraper Technique 升级版）

1. 找到竞品有大量外链的内容
2. 制作更优质、更新、更全面的版本
3. 主动 outreach（联系曾链接到竞品的网站）

**示例：** 竞品做了一份"2024年10款项目管理工具"，你做"2024年20款项目管理工具完整评测（附价格、功能、对比表）"

### 2. 资源页外链（Resource Page Link Building）

- 搜索 `intitle:"resources" + 关键词` 或 `intitle:"helpful resources" + 关键词`
- 找到资源页后，联系站长推荐你的资源
- 前提：你的资源确实值得被收录（工具、数据、指南）

### 3. 客座博客（Guest Posting）

- 找到相关行业的博客/媒体
- 投稿高质量文章，在作者简介中自然放置外链
- 避免低质量、纯外链交换的客座文章

### 4. 损坏链接建设（Broken Link Building）

1. 用 Ahrefs/Screaming Frog 找到竞品网站的失效外链
2. 创建对应主题的替代内容
3. 联系网站管理员报告失效链接并推荐你的内容

### 5. 摩天大楼技术（The Skyscraper Technique）

1. 找到竞品高外链文章
2. 制作至少 10x 更好的版本（更全、更深、更多数据）
3. 主动 outreach

### 6. HARO / Connectively（现为 Connectively）

- 记者发布采访请求（Queries）
- 回复请求，被引用时获得高权威外链
- 免费服务，需快速响应

### 7. 行业目录与点评网站

- DMOZ（已关闭）、行业黄页
- Yelp、Google Business Profile
- Crunchbase、LinkedIn Company Page
- Trustpilot、G2、Capterra（软件点评平台）

### 8. 新闻稿与 PR

- 真实新闻（融资、获奖、新产品发布）
- 通过 PR 服务或直接联系记者
- 避免低质量新闻稿（被视为垃圾）

## 外链质量评估

| 检查项 | 低质量信号 | 高质量信号 |
|--------|-----------|-----------|
| 域名权重 | DA/DR < 20 | DA/DR > 40 |
| 链接位置 | 页脚/侧边栏/广告区 | 正文内容区内 |
| 锚文本 | 完全匹配关键词堆砌 | 自然描述性锚文本 |
| 页面内容 | 内容稀少/无相关主题 | 与你的主题高度相关 |
| 链接页外链数 | 导出外链 > 100 | 导出外链 < 30 |
| 垃圾信号 | 大量广告、无实质内容 | 正常编辑内容 |

## 禁止行为（黑帽 SEO）

- ❌ 购买外链（Google 明确禁止，可能导致惩罚）
- ❌ 链接农场（大量站群互链）
- ❌ PBN（私有博客网络）
- ❌ 隐藏链接（CSS 隐藏）
- ❌ 论坛垃圾评论
- ❌ 自动化外链群发软件

## 外链监控

- **工具：** Ahrefs Alerts、Google Alerts、Moz Link Explorer
- 监控新外链：发现外链变化时检查是否为垃圾链接
- 丢失外链：分析原因（页面删除？对方撤链？），主动联系修复重要链接
- 定期清理低质量/垃圾外链（拒绝链接工具 Disavow）
