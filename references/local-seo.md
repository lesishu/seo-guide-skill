# 本地 SEO（Local SEO）

适用于有实体门店或提供本地服务的企业（餐厅、诊所、律所、家政等）。

## Google Business Profile（GBP）优化

GBP 是本地 SEO 的核心，必须优先完成。

### 创建与验证
1. 访问 [business.google.com](https://business.google.com)
2. 创建/认领商家页面
3. 完成验证（邮寄明信片/电话/邮件）

### 必填信息

| 字段 | 说明 |
|------|------|
| 商家名称 | 真实名称，不加关键词（避免违规）|
| 地址 | 精确到门牌号 |
| 服务范围 | 服务多个城市/区域时填写 |
| 类别 | 选择最准确的主类别 |
| 电话 | 本地固定电话优于手机号 |
| 网站 | 官网 URL |
| 营业时间 | 节假日特别时间也需标注 |
| 属性 | 设施、服务、支付方式等 |

### 提升 GBP 排名技巧

- **完整度：** 填写所有字段，Profile 完整度越高排名越好
- **类别选择：** 主类别选最精准的，添加最多 9 个次要类别
- **服务项目：** 详细列出所有服务（字数越多越好）
- **照片：** 定期上传店面、产品、服务人员照片（Google 数据显示高互动率）
- **帖子（Posts）：** 每周发布 1-2 次动态（活动、优惠、新品）
- **Q&A：** 主动添加常见问题与答案
- **评价：** 鼓励满意客户留下真实评价，及时回复所有评价

## 本地关键词策略

**核心关键词组合：**
```
[服务] + [地点]
例：朝阳区牙医、上海浦东搬家、北京海淀少儿英语

[服务] + [位置描述词]
例：望京附近餐厅、附近24小时药店
```

**Google 本地搜索特征：**
- 搜索含地名词时，显示本地包（Map Pack）+ 有机结果
- 无地名搜索（"牙医"）但用户有本地意图时，也会显示本地包
- 本地包 Top 3 = Google Maps 排名前三的商家

## 本地 SEO 信号（排名权重）

| 权重 | 信号 |
|------|------|
| 高 | Google Business Profile 完整度与相关性 |
| 高 | 评价数量、评分、评价质量 |
| 中高 | NAP（Name, Address, Phone）一致性 |
| 中 | 本地页面内容优化 |
| 中 | 本地外链（本地新闻、行业协会、社区网站）|
| 中 | 本地目录引用（黄页、行业网站）|
| 中 | 社交媒体本地互动 |
| 低 | 网站性能与移动端体验 |

## NAP 一致性

**NAP：** 企业名称（Name）、地址（Address）、电话（Phone）必须在全网一致。

**为什么要一致？**
Google 通过比对多个来源的 NAP 信息来验证商家真实性。

**需要保持一致的目录：**
- Google Business Profile
- 官网（Footer/联系页面）
- Yelp
- 百度地图 / 高德地图 / 腾讯地图
- 大众点评
- 行业目录（婚庆、医疗、家装等垂直平台）
- Facebook Page
- Apple Maps

**检查工具：** BrightLocal / Moz Local / Yext

## 本地目录管理

**主要平台：**
- Google Business Profile
- 百度地图、高德地图、腾讯地图（国内必做）
- 大众点评、口碑（国内必做）
- Yelp、TripAdvisor（跨境/旅游相关）
- Apple Maps Connect
- Facebook Business Page

**工具：** Yext、BrightLocal、Whitespark（批量管理本地目录）

## 本地内容策略

- 创建针对本地关键词的落地页（每个服务城市/区域一页）
- 内容包含：本地服务详情、本地案例、本地客户评价
- 在博客中引用本地事件/新闻（建立本地相关性）
- 本地新闻稿（本地媒体报道）

## 本地评价管理

**获取评价技巧：**
- 邮件/短信感谢信 + 链接直接跳转评价页
- 桌面/收银台放置评价提示卡（二维码）
- 简化的评价流程说明
- 在服务完成后 30 分钟内发送请求（趁热打铁）

**评价回复模板：**
```
感谢您的 5 星好评！[客户姓名]您好，很高兴为您提供满意的[服务类型]服务。
您的认可是我们最大的动力，[店铺名]团队会继续努力，期待下次为您服务！
```

**负面评价处理：**
- 24 小时内回复
- 表达歉意和理解
- 邀请私下沟通解决问题
- 切忌：反驳客户、删除差评（会让差评更突出）
